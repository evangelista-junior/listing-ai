## üöÄ Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

You need to have Node.js and a package manager (npm, yarn, pnpm, or bun) installed on your system.

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/evangelista-junior/listing-ai.git](https://github.com/evangelista-junior/listing-ai.git)
    cd listing-ai
    ```
2.  Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```
3.  **Environment Variables Setup:**
    Create a file named `.env.local` in the root of the project and add your necessary API keys and configuration settings.

    ```
    # Example:
    # NEXT_PUBLIC_AI_API_KEY=your_secret_key
    ```

### Running the Development Server

Execute the following command to start the application:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
